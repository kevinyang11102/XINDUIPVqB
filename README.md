# Python学生信息管理系统 python682

## 项目概述

本项目是基于Python的学生信息管理系统，旨在提供便捷的信息管理功能，主要包括登录注册、数据总览、学生信息管理、学生成绩管理、教师信息管理、考试信息管理和通知管理等模块。

## 技术栈

- **开发语言：** Python
- **框架：** Flask
- **前端技术：** HTML、CSS、JavaScript、jQuery
- **后端技术：** Python、Flask
- **数据库：** MySQL
- **开发工具：** PyCharm、Navicat

## 功能模块

- 登录注册
- 数据总览
- 学生信息管理
- 学生成绩管理
- 教师信息管理
- 考试信息管理
- 通知管理

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- **Python版本：** Python 3.0及以上
- **数据库版本：** MySQL 8.0

## 部署步骤

1. 确保Python 3.0以上及MySQL 8.0环境安装配置完成。
2. 克隆或下载项目至本地。
3. 在项目根目录下创建虚拟环境，并激活。
4. 安装依赖包：`pip install -r requirements.txt`
5. 配置数据库连接信息。
6. 运行应用：`python app/main.py`
